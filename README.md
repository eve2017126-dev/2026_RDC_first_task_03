# 💼 创业公司盈利预测 - 逻辑回归模型项目

## 📋 项目概述

本项目使用手写逻辑回归模型预测创业公司是否能够盈利。基于startup_data.csv数据集，通过深入的特征工程和模型实现，为创业投资决策提供数据支持。

### 🎯 任务要求
- **数据集分割** (3%): 选择合适的比例分割训练集和测试集
- **数据预处理和特征工程** (7%): 基于业务理解进行特征构造
- **手写逻辑回归算法** (7%): 实现逻辑回归模型的核心算法
- **模型评估** (3%): 使用分类问题的评估标准
- **数学原理理解和答辩** (20%): 深入理解算法原理

## 📊 数据集特征

### 原始特征
- **Startup Name**: 创业公司名称
- **Industry**: 行业领域（AI、金融科技、健康科技等）
- **Funding Rounds**: 融资轮数（1-5轮）
- **Funding Amount (M USD)**: 融资金额（百万美元）
- **Valuation (M USD)**: 公司估值（百万美元）
- **Revenue (M USD)**: 年收入（百万美元）
- **Employees**: 员工人数（5-5000人）
- **Market Share (%)**: 市场份额百分比
- **Profitable**: 是否盈利（1=盈利，0=不盈利）

### 构造的业务特征
- **融资效率**: 收入/融资金额
- **估值收入比**: 估值/收入（P/S比率）
- **人均收入**: 收入/员工数
- **融资轮次效率**: 融资金额/融资轮数
- **市场规模指数**: 市场份额 × 收入
- **资本效率指数**: (收入 × 市场份额) / 融资金额

## 🔧 技术实现

### 1. 数据预处理（重点7%）

#### 缺失值处理
- 数值特征：中位数填充
- 分类特征：众数填充

#### 特征工程
- **行业编码**: 频率编码和热度编码
- **业务特征构造**: 基于创业公司特点的特征设计
- **异常值处理**: IQR方法处理极端值

#### 数据标准化
- 使用StandardScaler进行Z-score标准化
- 确保特征尺度一致

### 2. 手写逻辑回归模型（核心7%）

#### 数学原理实现
```python
# Sigmoid函数
sigmoid(z) = 1 / (1 + e^(-z))

# 损失函数（对数损失）
J(θ) = -1/m * Σ [y*log(h(x)) + (1-y)*log(1-h(x))]

# 梯度计算
dJ/dθ_j = 1/m * Σ (h(x) - y) * x_j
```

#### 模型特点
- **梯度下降优化**: 自动学习最优参数
- **损失监控**: 实时显示训练进度
- **数值稳定性**: 防止数值溢出

### 3. 模型评估（3%）

#### 评估指标
- **准确率** (Accuracy): 整体预测正确率
- **精确率** (Precision): 盈利预测的准确性
- **召回率** (Recall): 盈利公司的识别率
- **F1分数**: 精确率和召回率的调和平均
- **AUC-ROC**: 模型整体区分能力

#### 可视化分析
- 混淆矩阵热力图
- ROC曲线
- 损失函数曲线
- 特征重要性图
