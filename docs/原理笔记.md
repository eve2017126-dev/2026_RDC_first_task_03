# 📚 逻辑回归数学原理笔记

## 🎯 核心概念

### 1. 逻辑回归的本质
逻辑回归是一种**分类算法**，用于解决二分类问题。虽然名字中有"回归"，但它实际上是分类算法。

**目标**: 预测创业公司是否盈利（1=盈利，0=不盈利）

### 2. 与线性回归的区别
| 特性 | 线性回归 | 逻辑回归 |
|------|----------|----------|
| 输出 | 连续值 | 概率值（0-1） |
| 应用 | 回归问题 | 分类问题 |
| 函数 | 线性函数 | Sigmoid函数 |

## 📈 数学原理详解

### 1. 假设函数 (Hypothesis Function)

**线性组合**:
```
z = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ = θᵀx
```

**Sigmoid函数**:
```
h(x) = σ(z) = 1 / (1 + e^(-z))
```

**数学解释**:
- 当 z → +∞ 时，h(x) → 1（盈利概率接近100%）
- 当 z → -∞ 时，h(x) → 0（盈利概率接近0%）
- 当 z = 0 时，h(x) = 0.5（50%概率）

### 2. 损失函数 (Loss Function)

**对数损失函数**:
```
J(θ) = -1/m * Σ [y⁽ⁱ⁾ * log(h(x⁽ⁱ⁾)) + (1-y⁽ⁱ⁾) * log(1-h(x⁽ⁱ⁾))]
```

**分情况理解**:
- 当 y=1 时：损失 = -log(h(x))
  - h(x)越大（接近1），损失越小
  - h(x)越小（接近0），损失越大
- 当 y=0 时：损失 = -log(1-h(x))
  - h(x)越小（接近0），损失越小
  - h(x)越大（接近1），损失越大

### 3. 梯度下降 (Gradient Descent)

**参数更新公式**:
```
θ_j := θ_j - α * ∂J(θ)/∂θ_j
```

**梯度计算**:
```
∂J(θ)/∂θ_j = 1/m * Σ (h(x⁽ⁱ⁾) - y⁽ⁱ⁾) * x_j⁽ⁱ⁾
```

**学习率 α 的作用**:
- α 太小：收敛慢，需要更多迭代
- α 太大：可能无法收敛，在最优解附近震荡

## 🔍 关键数学推导

### 1. Sigmoid函数导数
```
σ'(z) = σ(z) * (1 - σ(z))
```

**推导过程**:
```
令 σ(z) = 1/(1+e^(-z))
σ'(z) = d/dz [1/(1+e^(-z))]
      = e^(-z)/(1+e^(-z))²
      = [1/(1+e^(-z))] * [e^(-z)/(1+e^(-z))]
      = σ(z) * (1 - σ(z))
```

### 2. 损失函数梯度推导

**单个样本的损失**:
```
L(θ) = -[y*log(h) + (1-y)*log(1-h)]
```

**对 θ_j 求导**:
```
∂L/∂θ_j = ∂L/∂h * ∂h/∂z * ∂z/∂θ_j
```

**分步计算**:
1. ∂L/∂h = -[y/h - (1-y)/(1-h)]
2. ∂h/∂z = h*(1-h)  （Sigmoid导数）
3. ∂z/∂θ_j = x_j

**合并结果**:
```
∂L/∂θ_j = (h - y) * x_j
```

## 💻 代码实现对应

### 1. Sigmoid函数实现
```python
def sigmoid(self, z):
    # 防止数值溢出
    z = np.clip(z, -500, 500)
    return 1 / (1 + np.exp(-z))
```

### 2. 损失函数实现
```python
def compute_loss(self, y_true, y_pred):
    # 防止log(0)的情况
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    
    m = len(y_true)
    loss = -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))
    return loss
```

### 3. 梯度计算和参数更新
```python
# 计算梯度
dw = (1 / m) * np.dot(X.T, (y_pred - y))
db = (1 / m) * np.sum(y_pred - y)

# 更新参数
self.weights -= self.learning_rate * dw
self.bias -= self.learning_rate * db
```

## 🎯 业务应用中的数学理解

### 1. 特征权重的解释
- **正权重**: 特征值增加 → 盈利概率增加
- **负权重**: 特征值增加 → 盈利概率减少
- **绝对值大小**: 特征对结果的影响程度

### 2. 概率阈值选择
- **默认阈值**: 0.5
- **业务调整**: 根据误分类成本调整阈值
  - 如果漏掉盈利公司成本高 → 降低阈值
  - 如果误判不盈利公司成本高 → 提高阈值

### 3. 模型评估的数学基础

**混淆矩阵指标**:
- 准确率 = (TP+TN)/(TP+TN+FP+FN)
- 精确率 = TP/(TP+FP)
- 召回率 = TP/(TP+FN)
- F1分数 = 2*(精确率*召回率)/(精确率+召回率)

**AUC-ROC原理**:
- 真正率（TPR） = TP/(TP+FN)
- 假正率（FPR） = FP/(FP+TN)
- AUC = ROC曲线下的面积，衡量模型整体区分能力

## 📊 本项目中的具体应用

### 1. 特征工程与数学关系
- **融资效率** = 收入/融资金额
  - 反映资金使用效率，数学上体现为比值特征
- **估值收入比** = 估值/收入
  - 反映市场预期，数学上需要处理除零问题
- **人均收入** = 收入/员工数
  - 反映运营效率，数学上需要标准化

### 2. 数据预处理的数学考虑
- **标准化**: 使特征具有相同尺度，避免某些特征主导梯度下降
- **异常值处理**: 防止极端值对模型产生过大影响
- **缺失值填充**: 使用统计量保持数据分布

### 3. 模型训练的数学优化
- **学习率选择**: 通过实验找到合适的收敛速度
- **迭代次数**: 确保模型充分收敛但不过拟合
- **损失监控**: 实时观察训练过程，防止梯度爆炸
